# 考研网盘云群 H5 商城（纯静态版）

无需后端，直接打开就能用！

## 访问地址

```
https://kaede123456.github.io/-/
```

## 功能说明

- ✅ 店铺首页：双列商品卡片展示
- ✅ 个人中心：订单列表
- ✅ 支付流程：模拟支付 + 自动发货
- ✅ 数据存储：浏览器本地（localStorage）

## 使用方法

1. 打开网址即可使用
2. 点击商品购买
3. 点击「已完成支付」获取资料链接
4. 在个人中心查看订单

## 修改商品

编辑 `public/index.html` 中的 `GOODS_DATA` 数组即可修改商品信息。
